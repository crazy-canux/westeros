===================
pytestautomation
===================

Test Automation build with robot framework and selenium

=======
Install
=======

Use pip::

    $pip install pytestautomation

Use source code::

    $git clone https://github.com/crazy-canux/pytestautomation.git
    $cd pytestautomation
    $python setup.py install

===========
Quick Start
===========

Import this library::

    import pytestautomation
